## Installation

```bash
npm install --save-dev @hyeon/linter
```

## Usage

```js
import hyeonLinter from '@hyeon/linter'

export default [
  ...hyeonLinter.react,
  ...hyeonLinter.prettier,
  ...hyeonLinter.typescript,
]
```
